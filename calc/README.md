# Calculator MCP Server

Basic calculator MCP server built with FastMCP.

## Tools

- `add` - Addition
- `subtract` - Subtraction
- `multiply` - Multiplication
- `divide` - Division
- `power` - Exponentiation
- `sqrt` - Square root
- `mod` - Modulo

## Usage

```bash
make run      # Run server (stdio)
make inspect  # MCP Inspector
make http     # HTTP mode (port 8000)
```
